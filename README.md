# todastarefas.com.br [Controle de Tarefas]

Aplicação web de gerenciamento de tarefas pessoais construída com Flask.

## Tecnologias
- Python 3.12 / Flask 3.0
- SQLite + SQLAlchemy
- Bootstrap 5
- PythonAnywhere (hospedagem)

## Versão atual
V4 — Tema dark/light, busca por texto, calendário, notificações por e-mail e whatsapp

## Como rodar localmente
```bash
pip install -r requirements.txt
python app.py
```
Acesse: http://localhost:5050
